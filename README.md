# Rock Paper Scissors

A simple Rock Paper Scissors game built with React and Vite.

## Features

- Play Rock, Paper, or Scissors against the computer
- Tracks rounds, wins, losses, draws, and win streak
- Move history showing all previous rounds
- Reset button to start over

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## How to Play

1. Click **Rock**, **Paper**, or **Scissor** to make your move
2. The computer picks randomly
3. The result is shown along with updated stats
4. Click **Reset** to clear everything and start fresh

## Tech Stack

- React 19
- Vite
